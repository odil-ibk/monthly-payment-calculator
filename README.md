# Apartment Rent Calculator

This is my first personal project built with React.js.

The application calculates how much money should be paid based on the number of days a person has lived in an apartment.

## Features
- Enter number of days lived
- Enter total payment
- Simple and user-friendly interface

## Technologies Used
- React JS
- JavaScript
- HTML
- CSS

## Purpose
This project was created to practice React fundamentals and build my first independently owned project.
